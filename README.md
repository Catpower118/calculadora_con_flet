# calculadora basica en flet

una calculadora sencilla desarrollada en **python + flet**, soporte para:

- Suma
- Resta
- Multiplicación
- División
- Entrada por teclado
- Boton de borrado
- Evaluacion automatica con la tecla enter
- Interfaz limpia y responsiva

## Caracteristicas

- Interfaz grafica creada con flet
- Botones numericos y de operacion
- Soporte para:
  - Numeros
  - Operadores
  - Backspace para borrar
  - Enter para evaluar
- Boton **C** para borrar todo el contenido apliado (Ocupa todo el espacio)

## Instalacio 

clona este repositorio

## Tecnologias usadas

- Python 3
- Flet 0.80+
- JSON para configuracion de assets
